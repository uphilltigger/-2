＜使い方＞
<br>
<b>ウマ</b>と<b>運動強度</b>を選んで、給餌量を入力すれば充足率がわかるようになっています．
<br>(1)まずは現在の給餌量を入れて、なにがどのくらい満たされているのかを確認する．
<br>(2)主にエネルギー，タンパク質，総量，Ca/P比，Zn/Cu比に注意してそれぞれの値をいじる．
<br>(3)黒板に反映。後世のために記録もしておく。
<br>
<br>・注意点
<br>特別な配慮が必要な馬もいる．
<br>・クライム：喉詰まりを起こすのでビートはあげない． 
<br>・ノガロ：なぜかあまり増やさなくても痩せない．
<br>・パク：蹄が弱いので白サプリを入れること
<br>・高齢馬や胃腸が弱い馬：麦や濃厚ばかりだと消化管に負担大．ビートやキューブは消化に良いほか、ふすまは整腸作用があるのでこれらを重視したい。
<br>・新馬：乗馬クラブからなのか競走馬あがりなのかとかで変わってくるが、原則として体重が同じくらいの馬に合わせる．例：ワイルドヒロイン→セノーテ、ファーブルトン→ノガロ。食べ慣れない飼いは徐々に増やす．ふすまは特に残しやすい．
<br>・去勢手術が控えている馬（レア）：体力つけてあげないとかわいそうなので、気持ち多めに作って与えてあげた方がいいかも。
 
<br>	※入力値は一食あたり．<br>
<br>	※各値の表示：	青＝不足，緑＝理想，赤＝過剰
