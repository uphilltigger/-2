＜使い方＞
<br>
ウマと運動強度を選んで、給餌量を入力すれば充足率がわかるようになっています．<br>
		考え出したらキリが無いので、DE,CPと下の表の3つの値が問題なければ良しとしましょう。<br>  
		・入力値は一食あたり．<br>
		・各値の表示：	青＝不足，緑＝理想，赤＝過剰<br>
		<br>
		DE＝エネルギー<br>
		CP＝タンパク質<br>
		Zn=亜鉛：蹄の角質化に必要<br>
		VA（ビタミンA）：過剰に注意<br>
		VD（ビタミンD）：日光浴びれば大丈夫
